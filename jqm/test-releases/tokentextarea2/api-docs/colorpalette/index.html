Colorpalette

The color palette widget displays a palette of colors given a single color. It constructs the palette by calculating dividing the hue spectrum evenly around the hue of the given color into as many parts as there are palette entries and constructing one color out of each of the calculated hue values, including the hue of the given color, while using the given color’s saturation and lightness for the calculated colors.

Options

colorType: String

Default: null
The color that forms the basis for the palette. When you set the color, the colors option is set to a palette containing the color and hue-related siblings.

Code examples:

Initialize the colorpalette with the color option specified:

1
 
$( ".selector" ).colorpalette({ color: #f7b29a });

Get or set the color option, after initialization:

1
2
3
4
5
 
// getter
var color = $( ".selector" ).colorpalette( "option", "color" );
 
// setter
$( ".selector" ).colorpalette( "option", "color", #f7b29a );

colorsType: String

Default: #ff0000,#ff8000,#ffff00,#80ff00,#00ff00,#00ff80,#00ffff,#0080ff,#0000ff,#8000ff,#ff00ff,#ff0080
The colors to display in the palette. This is a comma-separated list of HTML RGB values. The number of colors in the palette will be adjusted to match the number of elements in the list. If the current color is the list, it will be selected.

Code examples:

Initialize the colorpalette with the colors option specified:

1
 
$( ".selector" ).colorpalette({ colors: #aabbcc,#ddeeff,#001122,#334455,#667788,#99aabb,#012345,#6789ab,#cdef01,#02468a,#13579b,#112358 });

Get or set the colors option, after initialization:

1

// setter
$( ".selector" ).colorpalette( "option", "colors", #aabbcc,#ddeeff,#001122,#334455,#667788,#99aabb,#012345,#6789ab,#cdef01,#02468a,#13579b,#112358 );

rowsType: Number

Default: 2
The number of rows in which to display the pallette.

Methods

destroy()Returns: jQuery (plugin only)

Destroys a colorpalette. This causes the DOM elemenent on which it is based to be restored to the state in which it was before the widget was created. Any DOM elements created as part of the widget will be destroyed.
  • This method does not accept any arguments.
Code examples:

Invoke the destroy method:

1
 
$( ".selector" ).colorpalette( "destroy" );

refresh()Returns: jQuery (plugin only)

Rebuilds the color palette.
  • This method does not accept any arguments.
Code examples:

Invoke the refresh method:

1
 
$( ".selector" ).colorpalette( "refresh" );

Triggered when the selected color changes.
