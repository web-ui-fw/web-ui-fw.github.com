Route Map

Routemap is a jquery mobile plugin to render a route data.

Routemap draws the lines between above two stations automatically, and it indicates a shortest or minimum transfer route between two stations that a user selected

Routemap supports JSON for DB ( routemap data ), so it’s possible to set DB by setting a file path or a variable on widget, and it allows us to modify each style of stations, routes, and labels easily

And also, it supports a multi-language.

1
2
 
<div data-role="routemap" data-db="data.json">
</div>

To create a routemap, create a container and add the data-role=”routemap” attribute. Using data-db attribute allows you to set a DB for the lines, stations, label and each style.

Options

dbType: String

Default: null

The URL of a JSON file or a javascript variable to load.

Code examples:

Initialize the routemap with the db option specified:

1
 
$( ".selector" ).routemap({ db: "data.json" });

Get or set the db option, after initialization:

1
2
3
4
5
 
// getter
var db = $( ".selector" ).routemap( "option", "db" );
 
// setter
$( ".selector" ).routemap( "option", "db", "data.json" );

languageType: String

Default: null

Determine the current languauge. The values must be the abbreviation of language names(ex: English > en )

Code examples:

Initialize the routemap with the language option specified:

1
 
$( ".selector" ).routemap({ language: "en" });

Get or set the language option, after initialization:

1
2
3
4
5
 
// getter
var language = $( ".selector" ).routemap( "option", "language" );
 
// setter
$( ".selector" ).routemap( "option", "language", "en" );

unitType: Integer

Default: 1

The length unit of the route map. If the value is 10 and a station coordinate is (1, 2)
then the station will be displayed at (10, 20).

Code examples:

Initialize the routemap with the unit option specified:

1
 
$( ".selector" ).routemap({ unit: 10 });

Get or set the unit option, after initialization:

1
2

4
5
 
// getter

// setter
$( ".selector" ).routemap( "option", "unit", 10 );

Methods

dishighlight( stations )Returns: jQuery (plugin only)

Removes a highlighting style from stations.
  • stations
    Type: Array
    The array of staion id
Code examples:

Invoke the dishighlight method:

1
 
$( ".selector" ).routemap( "dishighlight" );

getIdsByName( name )Returns: jQuery (plugin only)

Gets an array of station/line ids associated with the specified station name or line name.
  • name
    Type: String
    The name of station or line.
Code examples:

Invoke the getIdsByName method:

1
 
$( ".selector" ).routemap( "getIdsByName" );

getNameById( id )Returns: jQuery (plugin only)

Gets a string of station or line name with associated the specified station or line id.
  • id
    Type: String
    The id of station or line
Code examples:

Invoke the getNameById method:

1
 
$( ".selector" ).routemap( "getNameById" );

highlight( stations )Returns: jQuery (plugin only)

Add highlight styles to station or line.
  • stations
    Type: Array
    The array of staion id
Code examples:

Invoke the highlight method:

1
 
$( ".selector" ).routemap( "highlight" );

minimumTransfers( source, destination )Returns: jQuery (plugin only)

Returns an array of station ids which is the route having minimum transfers from source to destination.
  • source
    Type: String
    The staion id of source
  • destination
    Type: String
    The staion id of destination
Code examples:

Invoke the minimumTransfers method:

1
 
$( ".selector" ).routemap( "minimumTransfers" );

refresh( redraw )Returns: jQuery (plugin only)

Resizes&redraws the widget, after having made some external modifications.
  • redraw (default: false)
    Type: Boolean
    When set to true, the widget will redraw all of elements by a route data.
    Only re-layout the widget if set to false.
Code examples:

Invoke the refresh method:

1
 
$( ".selector" ).routemap( "refresh", options );

shortestRoute( source, destination )Returns: jQuery (plugin only)

Returns an array of station id which is the shortest route from source to destination.
  • source
    Type: String
    a station id of source
  • destination
    Type: String
    a station id of destination
Code examples:

Invoke the shortestRoute method:

1
 
$( ".selector" ).routemap( "shortestRoute" );

Events

create( event, ui )Type: routemapcreate

Triggered when a routemap is created
  • event
    Type: Event
  • ui
    Type: Object

Note: The ui object is empty but included for consistency with other events.

Code examples:

Initialize the routemap with the create callback specified:

1
2
3
 
$( ".selector" ).routemap({
  create: function( event, ui ) {}
});

Bind an event listener to the routemapcreate event:

1
 
$( ".selector" ).on( "routemapcreate", function( event, ui ) {} );

select( event, id )Type: routemapselect

Triggered when station or line are selected
  • event
    Type: Event
  • id
    Type: String
    selected station or line id

Note: The ui object is empty but included for consistency with other events.

Code examples:
