Virtualgrid

Displays an array of items tiled horizontally and vertically. The user can pan the list horizontally or vertically.

Note

You want to use a data on different domain server, you need to check whether to support CORS(Cross Origin Resources Sharing) in the server.

If the server does not support CORS, virtualgrid would not work properly.

Options

dataTypeType: String

Default: json
This attribute indicates the type of the referenced data. It supports ‘xml’ or ‘json’.

Code examples:

Initialize the virtualgrid with the dataType option specified:

1
 
$( ".selector" ).virtualgrid({ dataType: json });

Get or set the dataType option, after initialization:

1
2
3
4
5
 
// getter
var dataType = $( ".selector" ).virtualgrid( "option", "dataType" );
 
// setter
$( ".selector" ).virtualgrid( "option", "dataType", json );

directionType: String

Default: y
The axis along which the panning will take place. (note) Virtualgrid does not support both directions.

Code examples:

Initialize the virtualgrid with the direction option specified:

1
 
$( ".selector" ).virtualgrid({ direction: x });

Get or set the direction option, after initialization:

1
2
3
4
5
 
// getter

// setter
$( ".selector" ).virtualgrid( "option", "direction", x );

replaceHelperType: String

Default: null
If user want to process more efficiently what replacing DOM, User should define a function and set a function name to this attribute.

Code examples:

Initialize the virtualgrid with the replaceHelper option specified:

temlateType: String

Default: ''
The template.

Methods

refresh()Returns: jQuery (plugin only)

Rebuilds the virtual grid.
  • This method does not accept any arguments.
Code examples:

Invoke the refresh method:

1
 
$( ".selector" ).virtualgrid( "refresh" );

scrollTo( x, y )Returns: jQuery (plugin only)

Scrolls to the pair of coordinates specified by the parameters.
  • x
    Type: Number
    The x-coordinate to which to scroll.
  • y
    Type: Number
    The y-coordinate to which to scroll.
Code examples:

Invoke the method:

1
 
$( ".selector" ).virtualgrid( "scrollTo" );

Events

create( event, ui )Type: virtualgridcreate

Triggered when a virtualgrid widget is created.
  • event
    Type: Event
  • ui
    Type: Object

Note: The ui object is empty but included for consistency with other events.

Code examples:

Initialize the virtualgrid with the create callback specified:

1
2
3
 
$( ".selector" ).virtualgrid({
  create: function( event, ui ) {}
});

Bind an event listener to the virtualgridcreate event:

1
 
$( ".selector" ).on( "virtualgridcreate", function( event, ui ) {} );
