Token Text Area

Tokentextarea changes a text item to a button. It can be comprised of a number of button widgets. When a user types text and the text gets a specific event to change from a text to a button, the input text is changed to a Tokentextarea widget. A user can add the Tokentextarea widget to a contact list, email list, or another list. The typical use of this widget is composing a number of contacts or phone numbers in a specific area of the screen.

1
2
3
 
<div data-role="tokentextarea">
  <h1>Token Text Area</h1>
</div>

Options

descriptionType: String

Default: + {0}
Manage the message format. The message is displayed when the widget status changes to 'focus out'. : data-description="+ {0}".
Code examples:

Initialize the tokentextarea with the description option specified:

1
 
$( ".selector" ).tokentextarea({ description: fade });

Get or set the description option, after initialization:

1
2
3
4
5
 
// getter
var description = $( ".selector" ).tokentextarea( "option", "description" );
 
// setter
$( ".selector" ).tokentextarea( "option", "description", fade );

labelType: String

Default: To :

This option is to provide a label for a user-guide:data-label="To : ".

Code examples:

Initialize the tokentextarea with the label option specified:

// setter

linkType: String

Default: null
Set the ID of the page to link, like the contact. : data-link="pageId".

Methods

add( options )

If second argument does not exist, will insert to a new button at last position. Insert a new button at indexed position. The position is decided by the second argument. "index of position" means that the position of inserting a new button is decided by the second argument on "add" method. For example, if a user call the method like this "add("Tizen", 2)", new button labeled "Tizen" will be inserted on the third position.
  • options
    Type: Object
    • text (default: )
      Type: String
      The text ...........
    • index (default: )
      Type: Integer
      The index ...........
Code examples:

Invoke the add method:

1
 
$( ".selector" ).tokentextarea( "add" );

destroy()

destroy a tokentextarea (Remove all of the new DOM elements for the current widget that you created.)
  • This method does not accept any arguments.
Code examples:

Invoke the destroy method:

1
 
$( ".selector" ).tokentextarea( "destroy" );

focusIn()

Changes the focus status to 'focus in'. The status is able to manage a widget. 'Manage' means that user can add or remove a button.
  • This method does not accept any arguments.
Code examples:

Invoke the focusIn method:

1
 
$( ".selector" ).tokentextarea( "focusIn" );

focusOut()

Changes the focus status to 'focus out'. The status is not able to manage a widget. All buttons that contained in the widget are removed and summarized message is displayed.
  • This method does not accept any arguments.
Code examples:

Invoke the focusOut method:

1
 
$( ".selector" ).tokentextarea( "focusOut" );

inputText( options )

If no argument exists, gets the string from the input box. If the argument exists, sets the string to the input box.
  • options
    Type: Object
    • text (default: )
      Type: String
      The text ...........
Code examples:

Invoke the inputText method:

1
 
$( ".selector" ).tokentextarea( "inputText", options );

length()

Gets the number of the buttons.
  • This method does not accept any arguments.
Code examples:

Invoke the length method:

1
 
$( ".selector" ).tokentextarea( "length" );

remove( options )

If no argument exists, all buttons are removed. Remove a button at indexed position. The position is decided by the second argument. (index: index of button)
  • options
    Type: Object
    • index (default: )
      Type: Integer
      The index ...........
Code examples:

Invoke the remove method:

1
 
$( ".selector" ).tokentextarea( "remove" );

select( options )

If no argument exists, gets a string of the selected block. If any button isn't selected on a tokentextarea widget, this method returns "null" value. When a user call this method with an argument which is a number type, this method selects the button which is matched with the argument.
  • options
    Type: Object
    • index (default: )
      Type: Integer
      The index ...........
Code examples:

Invoke the select method:

1
 
$( ".selector" ).tokentextarea( "select" );

Events

create( event, ui )Type: tokentextareacreate

Triggered when a tokentextarea is created
  • event
    Type: Event
  • ui
    Type: Object
Code examples:

2
