jQuery Mobile Web UI Framework API Documentation

The Web UI Framework is a set of widgets and other components for jQuery Mobile.

To get started, view the full listing of entries.

This site provides API documentation for the widgets of the Web UI Framework.

The Web UI Framework supports jQuery Mobile 1.3.0 to 1.3.1, and jQuery 1.7.0 to 1.9.1.

To download the Web UI Framework, first download jQuery Mobile. Then, add the Web UI Framework JS and CSS to a Web page after you have already added jQuery Mobile. The Web UI Framework JS and CSS files can be found here:

CSS http://web-ui-fw.github.io/jqm/0.2.0/web-ui-fw.css
JS http://web-ui-fw.github.io/jqm/0.2.0/web-ui-fw.js
