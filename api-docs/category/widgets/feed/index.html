Colorpalette

The color palette widget displays a palette of colors given a single color. It constructs the palette by calculating dividing the hue spectrum evenly around the hue of the given color into as many parts as there are palette entries and constructing one color out of each of the calculated hue values, including the hue of the given color, while using the given color's saturation and lightness for the calculated colors.

Options

colorType: String

Default: null
The color that forms the basis for the palette. When you set the color, the colors option is set to a palette containing the color and hue-related siblings.
Code examples:

Initialize the colorpalette with the color option specified:

1
 
$( ".selector" ).colorpalette({ color: #f7b29a });

Get or set the color option, after initialization:

1
2
3

5

// setter
$( ".selector" ).colorpalette( "option", "color", #f7b29a );

colorsType: String

Default: #ff0000,#ff8000,#ffff00,#80ff00,#00ff00,#00ff80,#00ffff,#0080ff,#0000ff,#8000ff,#ff00ff,#ff0080
The colors to display in the palette. This is a comma-separated list of HTML RGB values. The number of colors in the palette will be adjusted to match the number of elements in the list. If the current color is the list, it will be selected.

HSVPicker

The Hue/Saturation/Value picker displays three sliders - one for each channel. Whenever the user drags one of the sliders, the resulting colour is calculated and assigned to the widget's "color" option. The "colorchanged" signal is emitted. In addition, if the widget is based on a form input, the "change" signal is triggered on the input.

Token Text Area

Tokentextarea changes a text item to a button. It can be comprised of a number of button widgets. When a user types text and the text gets a specific event to change from a text to a button, the input text is changed to a Tokentextarea widget. A user can add the Tokentextarea widget to a contact list, email list, or another list. The typical use of this widget is composing a number of contacts or phone numbers in a specific area of the screen.

1
2
3

</div>
